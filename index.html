RPS-Multiplayer

Rock, Paper, Scissors Online Multiplayer! Two players are able to play at any time by joining either player 1 or player 2 with a name input. The center of the site features a chat box for the two players to talk to each other as well as for the site to display who has won. Wins and losses for each side are shown below each player's name and are reset to 0 upon a player leaving the game. To leave the game players can simply close the window the game is being played in which will enable other players to join in their place. Player 2 always goes after Player 1 and only the players can see their button options(other users can simply observe and wait for an open player space).

Liri Bot (Node App)

This program is a simple liri-bot used to search for concerts using Bands in Town API, songs using Node Spotify API, and movies using OMDb API. Once launched the program asks the user what they want to do and depending upon their selection asks what the name of the concert/song/movie is they wish to search for and then uses API's and npm packages to retrieve the desired information and display it to them with some ASCII art flare. After the information is provided the program stores the display results inside log.txt and then promts the user what they want to do next.

Bamazon (Node App)

Bamazon is a mock storefront that uses MySQL to store and modify inventory of products across various departments. Users can either use bamazonCustomer.js, bamazonManager.js, or bamazonSupervisor.js in order to modify or check the database. Every time the user is asked for input their response is validated using Inquirer's validate function in order to avoid database errors, such as inputing a non-number or an empty response. The app in its entirety runs in the console/terminal.

Friend Finder

A site to find friends, Friend Finder! The website takes in user input and then lets them know who the best match for them is. The user is then added to an array in the back-end in order to improve future matches with new users. There is no database being used to store user information, only a simple javascript array. Bark

Bark! Voice what's going on around you on our location based social platform. New users can create an account by entering a valid email, a password, a username and hitting signup. Returning users can enter the same three fields and hit login to be validated. Once logged a user must allow the site to access their location in order to grab posts in the local area. From the home page users can view the latest posts from all categories or filter them by category as well as make posts and go to thier own profile page to see all of their posts. Making posts is simple and straight forward, simply fill out the required fields and Bark!
